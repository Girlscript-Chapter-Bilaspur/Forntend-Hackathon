About

Who are we?

We are considered as a chapter of Girlscript India foundation, which is a section-8 non profit company, registered under the government of India on the 2nd June, 2017. Our aim is to spread awareness about technology and to provide a platform for everyone to generate an identity. With some innovative ideas our core team has 2 Chapter Leaders with 1 community manager, 6 leads, 8 coordinators and 23 members. We are looking forward to an awesome experience in learning new skills and building a better community.


Our Event

We, at Girlscript Bilaspur, have planned a City Level Hackathon(Frontend Technologies, for beginners) which is our first event. We are conducting this Hackathon for those who have the potential to usher their imagination to reality. This Hackathon is going to be out of the ordinary, even beginners can join to engage themselves in something productive. We will be using GITHUB repository as a hackathon conducting platform, where a participant has to maintain a private directory and make a Pull Request(PR) to the master repository. The PR will have a basic template which will be provided on the fifth day of the Hackathon in the second session.

Schedule

6 introductory sessions + 6 hands-on sessions

  • July 12

    Basics of Web Development and Basics of HTML

    (11AM-12PM)

    This session will revolve around the basics of web, ranging from what browser,server,internet are. We will be giving a brief overview of how structuring of webpage is done using HTML.
    [Hands On Evening Session(4PM - 5PM)]

  • July 13

    Basics of Styling and CSS

    (11AM-12PM)

    Style and Design are two main aspects of web development, that gives a life to the structure of the webpage. In this session you will learn about the styling and designing using CSS(Cascasing Style Sheets) by linking that to the main HTML code.
    [Hands On Evening Session(4PM - 5PM)]

  • July 14

    Bootstrap Library

    (11AM-12PM)

    Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components. This session will give a brief overview of the usages of bootstrap.
    [Hands On Evening Session(4PM - 5PM)]

  • July 15

    Basics of Javascript

    (11AM-12PM)

    How about giving some functionality to your HTML elements ? You may want your elements to rotate, to swing or to dance. Javascript is the thing that makes it possible. This session will teach you the basics as well as will show you some of the important aspects of Javascript in website development.
    [Hands On Evening Session(4PM - 5PM)]

  • July 16

    Basics of GIT/GITHUB

    (11AM-12PM)

    Ever heared about VCS (Version Control System) and how that works ? GITHUB provides hosting for software development and version control using GIT. This session will give you a complete basic overview of GITHUB as well as give you an idea on how to upload your project in the main repository in this Hackathon.
    [Hands On Evening Session(4PM - 5PM)]

  • July 17

    Designing and Entire Web App Demonstration

    (11AM-12PM)

    Before coding any product, designning is a must. In this session we will be discussing about the wireframing Techniques and Designing using Adobe Xd and Canva as our tools. Also there is an evening session where the entire demonstration of a web app will be shown along with the explanation.

  • Hackathon
    Begins!

    July 18th - 19th

    48 Hours Long Hackathon

    Each participant must participate in this 48 hours long hackathon. We will be dividing the entire hackathon into four segments of 12 hours each, where a problem statement will be provided. Participant has to submit some demonstration of their progress every 12 hours.
